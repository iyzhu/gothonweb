# gothonweb

**gothonweb** is a simple python game on the web.

### Installation
**Requirements** `flask`
```
pip3 install flask
```
**Clone from GitHub**
```
git clone https://github.com/tsmol07/gothonweb.git
```

### Usage
```bash
cd gothonweb
python3 app.py
```
Play it on `http://127.0.0.1:5000`

### Running tests

Use pip to install `nose`
```bash
pip3 install nose
```

Run nosetests in bash/zsh
```bash
nosetests
```
